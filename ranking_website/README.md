# Classificação dos candidatos

Um site pequeno feito para ler os dados de resultados das provras e ordernar de acordo com algum campo (normalmente a nota final). O objetivo é facilitar a leitura dos resultados apresentando a ordem de classificação (a não a ordem alfabética que normalmente publicam).

Acesse o site aqui: https://hacktestes.github.io/Study/ranking_website/src/rank_candidates.html
